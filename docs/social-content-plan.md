# Social Media Content Plan — CrewCircle

## Strategic Positioning

CrewCircle is a **boutique B2B AI consultancy** for Australian small businesses. The brand voice is:
- Down-to-earth, no jargon
- Practical ("stuff that works", not theory)
- Australian vernacular ("sorted", "no dramas")
- Privacy-first, local-first

**Primary platform:** LinkedIn (97% of competitor firms use it)
**Secondary:** GitHub (developer audience for tooling credibility)
**Tertiary (future):** YouTube (if video content pipeline develops)

---

## Cross-Posting Strategy

### Blog → LinkedIn
Every blog post published on crewcircle.com.au gets:
1. A LinkedIn post within 24hrs with:
   - 2-3 sentence summary in the post body
   - Link to the full article
   - 1 key metric or takeaway as a graphic card
2. Tag relevant clients or collaborators (with permission)

### GitHub → LinkedIn
When a new tool/repo is published:
1. Post on LinkedIn with:
   - What it does (1 sentence)
   - Link to GitHub
   - "Built by CrewCircle" tagline

## Visual Style Guide

**Logo:** CrewCircle logo (invert for dark backgrounds)
**Colours:** Match website palette (orange accent, slate/neutral primary)
**Typography:** Clean, large numbers for metrics, sans-serif body
**Image style:** Screenshots of real tools, desk photos, simple graphics
**Avoid:** Stock photos of "people pointing at whiteboards"

## Repurposing Flow

```
Blog Post
  ├── LinkedIn summary post (immediate)
  ├── Key stat as single-image card (1 week later)
  ├── LinkedIn poll/engagement post (2 weeks later)
  └── Newsletter roundup mention (quarterly)

New Tool/App
  ├── GitHub repo with README
  ├── LinkedIn launch post
  ├── Case study blog post (1 month later)
  └── Metrics card for LinkedIn (2 months later)
```
